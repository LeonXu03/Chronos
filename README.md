# Discord bot setup with Python tutorial I used

# Bot requirements
1. Administrator scope (follow tutorial above)
2. Enable 'Privileged Gateway Intents' in Discord Developer Portal on your Bot application (follow tutorial above). This makes it so Bot can read messages, see user voice status, etc. Needed for proper bot functionality.

# Imports needed (if anything else pops up, a quick Google search will help)
python3 -m pip install -U discord.py
pip install pytz

# Run 
python chronos.py OR python3 chronos.py

# Terminate program
Windows/Linux: [Ctrl + C] OR [Ctrl + \]
Mac: [Command + \]
